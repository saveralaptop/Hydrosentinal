# TT Norms Pro Font Files

Add the following font files to this directory:

- `tt-norms-pro-regular.woff2` (weight 400)
- `tt-norms-pro-semibold.woff2` (weight 600)

These files are referenced in `src/index.css` and will be loaded via @font-face declarations.

Once you add these files, the landing page will use TT Norms Pro as the primary font family across all text.
